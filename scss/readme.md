# SCSS

本のデザインを行います。

 * app.scss
   * このファイルに関連しているファイルをビルドして使用します。
 * base.scss
   * 基本的な機能のデザイン
 * column.scss
   * コラムのデザイン
 * counter.scss
   * 第〇章といったカウントをするようなデザイン 
 * cover.scss
   * 表紙デザイン
 * font.scss
   * フォント情報
 * media.scss
   * 紙印刷時のデザイン
 * q_and_a.scss
   * Q&Aのデザイン
 * talk.scss
   * 吹き出しデザイン
 * toc.scss
   * 目次デザイン
 * imprint.scss
   * 奥付デザイン
 * profile.scss
   * プロフィールデザイン
 * variables.scss
   * 変数
